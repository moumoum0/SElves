## 功能特性

- **交流**：系统成员之间的实时消息沟通
- **待办**：共享任务列表与进度追踪
- **投票**：集体决策投票功能
- **现代化界面**：Material 3 设计，支持深色/浅色主题
- **数据本地化**：所有数据存储在本地，保护隐私

## 快速开始

### 前提条件

- [Android Studio](https://developer.android.com/studio) (最新稳定版)
- JDK 11+
- Android SDK (API 26+)

### 安装与运行

1. 克隆仓库
   ```sh
   git clone https://github.com/moumoum0/SElves.git
   ```
2. 使用 Android Studio 打开项目
3. 编译并安装到设备
   ```sh
   ./gradlew installDebug
   ```
